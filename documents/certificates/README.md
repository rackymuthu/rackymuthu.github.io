# Certificates Organization

This folder contains all professional certificates and achievements organized by category.

## Folder Structure

### `/technical/`
- Cloud certifications (AWS, Azure, GCP)
- Programming certifications (Python, Java, etc.)
- Technology-specific certifications (Docker, Kubernetes, etc.)
- Framework certifications (Django, React, etc.)
- AI/ML certifications (Databricks, etc.)

#### Current Certificates:
- **2025-09-23_Databricks_Generative-AI-Fundamentals.pdf** - Databricks Generative AI Fundamentals certification

### `/professional/`
- Project management certifications (PMP, Agile, Scrum)
- Leadership and management certificates
- Industry-specific professional certifications
- Consulting and client management certificates

### `/education/`
- Academic transcripts
- Degree certificates
- Course completion certificates
- Training program certificates

## File Naming Convention

Use the following naming convention for consistency:

```
[YYYY-MM-DD]_[Organization]_[CertificateName].[ext]
```

Examples:
- `2024-01-15_AWS_Solutions-Architect-Associate.pdf`
- `2023-06-20_Deloitte_Leadership-Excellence.pdf`
- `2022-11-10_Coursera_Python-Specialization.pdf`

## Adding New Certificates

1. Determine the appropriate category folder
2. Use the naming convention above
3. Update the main portfolio if it's a significant certification
4. Keep original file names as backup if needed

## Notes

- All certificates should be in PDF format when possible
- Keep file sizes reasonable for web deployment
- Update the main portfolio website when adding major certifications
